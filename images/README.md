# /images — куда класть фото готовых изделий

Каждый слот на лендинге ждёт файл с конкретным именем. Положи файлы в эту папку с указанными именами — плейсхолдеры на странице автоматически перекроются твоими фото. Имя файла видно на самой подсказке в плейсхолдере (📷 рядом с подписью).

Рекомендации по съёмке: чистый/однотонный фон, кадр вертикальный для всех «изделий», горизонтальный — для «лукбука» и «команды». Размер по длинной стороне 1600–2000 px, JPG 80–90% качества.

## Список файлов

### Hero (главный экран)
- `hero-main.jpg` — флагманское изделие, формат **4:5** (например 1200×1500). Тон тёмный, можно с цветным фоном.

### Фото-полоса под hero
- `strip-1.jpg` — футболка чёрная, **4:5**
- `strip-2.jpg` — худи оверсайз, **4:5**
- `strip-3.jpg` — лонгслив с печатью, **4:5**
- `strip-4.jpg` — макро: вышивка/шеврон, **3:4**

### Блок «Что шьём»
- `tee-main.jpg`    — флагманская футболка, вертикаль
- `hoodie-main.jpg` — худи (можно на светлом фоне — слот настроен под светлый)
- `long-main.jpg`   — лонгслив
- `detail-main.jpg` — макро печати/вышивки

### Галерея «Изделия в кадре»
- `gallery-01.jpg` — обложка, **4:5** (большой)
- `gallery-02.jpg` — футболка
- `gallery-03.jpg` — деталь печати (светлый фон ок — слот светлый)
- `gallery-04.jpg` — худи со спины
- `gallery-05.jpg` — лонгслив
- `gallery-06.jpg` — лукбук в полный рост (большой кадр)
- `gallery-07.jpg` — вышивка / ярлык (светлый фон ок)
- `gallery-08.jpg` — групповое (команда / мероприятие), **горизонталь**
- `gallery-09.jpg` — упаковка / box

### Кейсы
- `case-01.jpg` — корпорация (главный кейс)
- `case-02.jpg` — фестиваль / open-air
- `case-03.jpg` — FMCG дроп
- `case-04.jpg` — спортивная форма
- `case-05.jpg` — welcome-набор

## Если хочешь поменять имя файла
Открой `index.html`, найди нужный блок по подписи (например, `data-label="hero-main.jpg"`) и поменяй имя и в `data-label`, и в `<img src="images/...">`.
